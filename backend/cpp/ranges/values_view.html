namespace views {     inline constexpr auto values = ranges::elements<1>; } (2) (since C++20) Takes a view of tuple-like values (e.g. std::tuple or std::pair), and produces a view with a value-type of the second element of the adapted view's value-type. 1) An alias for ranges::elements_view. 2) RangeAdaptorObject (and also RangeAdaptorClosureObject). The expression views::values(e) is expression-equivalent to values_view>{e} for any suitable subexpression e.
