default; (1) (since C++20) constexpr explicit drop_view( V base, ranges::range_difference_t count ); (2) (since C++20) Constructs a drop_view. 1) Default constructor. Value-initializes the underlying view base_ and initializes the count count_ to ​0​. After construction, base() returns a copy of V() and size() equals to the size of the underlying view. 2) Initializes the underlying view base_ with std::move(base) and the count count_ with count. After construction, base() returns a copy of base and size() returns ranges::size(base) - count if the size of base is not less than count, or ​0​ otherwise.
