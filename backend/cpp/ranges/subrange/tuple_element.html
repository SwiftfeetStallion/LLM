(4) (since C++20) The partial specializations of std::tuple_element for std::ranges::subrange provide compile-time access to the iterator or sentinel type of a subrange, using tuple-like syntax. They are provided for structured binding support. 1,2) Obtains the iterator type, i.e. I. 3,4) Obtains the sentinel type, i.e. S. If the subrange is volatile-qualified, the result types are also volatile-qualified because the partial specialization for volatile or const volatile types are used. Such usage is deprecated.
