range_adaptor_closure {}; (since C++23) std::ranges::range_adaptor_closure is a helper class template for defining a RangeAdaptorClosureObject. Let t be the object of type T, the implementation ensures that t is a range adaptor closure object if all the requirements are met: t is a unary function object that takes one range argument. T has exactly one public base class ranges::range_adaptor_closure, and T has no base classes of type ranges::range_adaptor_closure for any other type U. T does not satisfy range.
