Given the subexpression of which t denotes the (possibly materialized) result object as E: If ranges::size(t) is ill-formed, ranges::ssize(E) is also ill-formed. Otherwise, let Signed be make-signed-like-t ﻿: If std::ptrdiff_t is wider than Signed, ranges::ssize(E) is expression-equivalent to static_cast(ranges::size(t)). Otherwise, ranges::ssize(E) is expression-equivalent to static_cast(ranges::size(t)).
