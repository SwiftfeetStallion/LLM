constexpr auto end() const; (since C++20) Returns an iterator or a sentinel representing the end of the resulting subrange. Equivalent to: constexpr auto end() {     if constexpr (ranges::common_range)         return iterator{*this, ranges::end(base_), {}};     else         return sentinel{*this}; } [edit] Return value An iterator or a sentinel. [edit] Example Run this code #include #include #include   int main() { constexpr std::string_view keywords{"bitand bitor bool break"}; std::ranges::split_view kw{keywords, ' '}; assert(4 == std::ranges::distance(kw.begin(), kw.end())); } [edit] See also begin returns an iterator to the beginning (public member function) [edit] end returns an iterator or a sentinel to the end (public member function of std::ranges::lazy_split_view) [edit] ranges::end(C++20) returns a sentinel indicating the end of a range(customization point object)[edit]
