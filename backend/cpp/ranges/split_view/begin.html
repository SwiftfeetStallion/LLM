constexpr /*iterator*/ begin(); (since C++20) Returns an iterator to the first found subrange. In order to provide the amortized constant time complexity required by the range concept, this function caches the result within the split_view (by means of cached_begin_ member) for use on subsequent calls. Let base_ be the underlying data member. Equivalent to: constexpr /*iterator*/ begin() { if (!cached_begin_.has_value()) cached_begin_ = this->find_next(ranges::begin(base_)); return {*this, ranges::begin(base_), cached_begin_.value()}; } Contents 1 Return value 2 Complexity 3 Example 4 See also [edit] Return value An iterator.
