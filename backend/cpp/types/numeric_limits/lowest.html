numeric_limits::signaling_NaN numeric_limits::denorm_min Helper types float_round_style float_denorm_style [edit]  static constexpr T lowest() noexcept; (since C++11) Returns the lowest finite value representable by the numeric type T, that is, a finite value x such that there is no other finite value y where y < x. This is different from std::numeric_limits::min() for floating-point types. Only meaningful for bounded types.
