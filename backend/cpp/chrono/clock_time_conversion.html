It does so by providing a const-callable operator() that accepts an argument of type std::chrono::time_point and returns a std::chrono::time_point that represents an equivalent point in time. The duration of the returned time point is computed from the source duration in a manner that varies for each specialization. clock_time_conversion is normally only used indirectly, via std::chrono::clock_cast. A program may specialize clock_time_conversion if at least one of the template parameters is a user-defined clock type. The primary template is an empty struct.
