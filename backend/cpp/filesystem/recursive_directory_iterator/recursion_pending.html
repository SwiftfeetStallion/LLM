begin(std::filesystem::recursive_directory_iterator)end(std::filesystem::recursive_directory_iterator) [edit]  bool recursion_pending() const; (since C++17) Returns true if the next increment will cause the directory currently referred to by *this to be iterated into. This function returns true immediately after construction or an increment. Recursion can be disabled via disable_recursion_pending().
