directory_entry& d ); (since C++17) Performs stream output on the directory entry d. Equivalent to return os << d.path();. This function template is not visible to ordinary unqualified or qualified lookup, and can only be found by argument-dependent lookup when std::filesystem::directory_entry is an associated class of the arguments. This prevents undesirable conversions in the presence of a using namespace std::filesystem; using-directive.
