This C-compatibility header is part of the general utility library, in particular, it provides several function templates to access, manipulate, and process individual bits and bit sequences. The byte ordering (endianness) of scalar types can be inspected via __STDC_ENDIAN_NATIVE__ facility. It is unspecified whether provides any declarations in namespace std.
