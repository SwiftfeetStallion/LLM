If n == 0, the position of the first character of the entire matched expression is returned. If n > 0 && n < size(), the position of the first character of the nth sub-match is returned. if n >= size(), a position of the first character of the unmatched match is returned. ready() must be true. Otherwise, the behavior is undefined.
