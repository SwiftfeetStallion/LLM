If n == 0, a string representing entire matched expression is returned. If 0 < n && n < size(), a string representing nth sub-match is returned. if n >= size(), a string representing the unmatched match is returned. The call is equivalent to string_type((*this)[n]); ready() must be true. Otherwise, the behavior is undefined.
