If n == 0, returns a reference to the std::sub_match representing the part of the target sequence matched by the entire matched regular expression. if n >= size(), returns a reference to a std::sub_match representing an unmatched sub-expression (an empty subrange of the target sequence). ready() must be true. Otherwise, the behavior is undefined.
