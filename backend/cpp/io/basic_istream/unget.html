First, clears eofbit. Then,(since C++11) unget behaves as UnformattedInputFunction. After constructing and checking the sentry object, if any ios_base::iostate flags are set, the function sets failbit and returns. Otherwise, calls rdbuf()->sungetc(). If rdbuf()->sungetc() returns Traits::eof(), calls setstate(badbit). In any case, sets the gcount() counter to zero. If an internal operation throws an exception, it is caught and badbit is set. If exceptions() is set for badbit, the exception is rethrown.
