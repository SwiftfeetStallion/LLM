Behaves as UnformattedOutputFunction (except without actually performing output). After constructing and checking the sentry object, (since C++11) If fail()==true, returns pos_type(-1). Otherwise, returns rdbuf()->pubseekoff(0, std::ios_base::cur, std::ios_base::out).
