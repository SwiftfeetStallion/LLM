The behavior is undefined if s is not pointing at an element of a null-terminated array. 3,4) Constructs the underlying std::strstreambuf by calling strstreambuf(s, n) and initializes the base class with the address of the strstreambuf. The behavior is undefined if s is not pointing at an element of an array whose length is at least n elements.
