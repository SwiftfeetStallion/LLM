The wrapped std::basic_spanbuf object is constructed as basic_spanbuf(s, mode). 2) Move constructor. Move constructs the std::basic_iostream base subobject and the wrapped std::basic_spanbuf from those of rhs, and then calls set_rdbuf with the address of the wrapped std::basic_spanbuf in *this to install it. 3) Copy constructor is deleted. basic_spanstream is not copyable.
