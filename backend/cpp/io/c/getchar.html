Equivalent to std::getc(stdin). Contents 1 Parameters 2 Return value 3 Example 4 See also [edit] Parameters (none) [edit] Return value The obtained character on success or EOF on failure. If the failure has been caused by end of file condition, additionally sets the eof indicator (see std::feof()) on stdin. If the failure has been caused by some other error, sets the error indicator (see std::ferror()) on stdin. [edit] Example std::getchar with error checking.
