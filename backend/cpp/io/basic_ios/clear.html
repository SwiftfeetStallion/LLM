By default, assigns std::ios_base::goodbit which has the effect of clearing all error state flags. If rdbuf() is a null pointer (i.e. there is no associated stream buffer), then state | std::ios_base::badbit is assigned. It can be a combination of the following constants: Constant Explanation goodbit no error badbit irrecoverable stream error failbit input/output operation failed (formatting or extraction error) eofbit associated input sequence has reached end-of-file [edit] Return value (none) [edit] Exceptions If the new error state includes a bit that is also included in the exceptions() mask, throws an exception of type failure.
