These function templates are used when the left operand is not a basic_const_iterator. I satisfies the exposition-only concept /*not-a-const-iterator*/ if and only if it's not a specialization of basic_const_iterator. These functions are not visible to ordinary unqualified or qualified lookup, and can only be found by argument-dependent lookup when basic_const_iterator is an associated class of the arguments.
