operator++(); (1) istream_iterator operator++( int ); (2) Reads a value from the underlying stream (using its operator>>) and stores it into the iterator object. If the read fails (the underlying stream's fail() returns true), the iterator becomes the end-of-stream iterator. The behavior is undefined if the iterator is end-of-stream iterator.
