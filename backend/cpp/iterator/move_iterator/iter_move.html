the underlying iterator to its associated rvalue reference type. Equivalent to return std::ranges::iter_move(i.base());. This function template is not visible to ordinary unqualified or qualified lookup, and can only be found by argument-dependent lookup when std::move_iterator is an associated class of the arguments.
