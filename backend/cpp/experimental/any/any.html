(1) (library fundamentals TS) any( const any& other ); (2) (library fundamentals TS) any( any&& other ) noexcept; (3) (library fundamentals TS) template     any( ValueType&& value ); (4) (library fundamentals TS) Constructs a new any object. 1) Constructs an empty object. 2,3) Copies (2) or moves (3) content of other into a new instance, so that any content is equivalent in both type and value to those of other prior to the constructor call, or empty if other is empty. 4) Constructs an object with initial content an object of type std::decay_t, direct-initialized from std::forward(value). If std::is_copy_constructible>::value is false, the program is ill-formed. This overload participates in overload resolution only if std::decay_t is not the same type as any.
