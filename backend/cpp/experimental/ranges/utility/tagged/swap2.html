This function is not visible to ordinary unqualified or qualified lookup, and can only be found by argument-dependent lookup when tagged is an associated class of the arguments. [edit] Notes Unlike most non-member swaps in the standard library, this swap is a hidden friend function declared in the body of tagged, not a function template.
