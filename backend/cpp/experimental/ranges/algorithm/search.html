Contents 1 Parameters 2 Return value 3 Complexity 4 Possible implementation 5 Example 6 See also [edit] Parameters first1, last1 - the range of elements to examine r1 - the range of elements to examine first2, last2 - the range of elements to search for r2 - the range of elements to search for pred - predicate to apply to the projected elements proj1 - projection to apply to the elements in the first range proj2 - projection to apply to the elements in the second range [edit] Return value An iterator to the beginning of first occurrence of the sequence [first2, last2) in the range [first1, last1). If [first2, last2) is empty, first1 is returned. If no such occurrence is found, an iterator that compares equal to last1 is returned. [edit] Complexity At most S * N applications of the predicate and each projection, where S = last2 - first2 and N = last1 - first1.
