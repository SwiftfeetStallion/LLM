Contents 1 Parameters 2 Return value 3 Remarks 4 Exceptions 5 See also [edit] Parameters m, n - integer values [edit] Return value If both m and n are zero, returns zero. Otherwise, returns the greatest common divisor of |m| and |n|. [edit] Remarks If either M or N is not an integer type, or if either is (possibly cv-qualified) bool, the program is ill-formed. If either |m| or |n| is not representable as a value of type std::common_type_t, the behavior is undefined. [edit] Exceptions Throws no exceptions.
