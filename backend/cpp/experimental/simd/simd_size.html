If T is a vectorizable type and is_abi_tag_v is true, provides the member constant value equal to the width (number of elements) of simd, irrespective of whether simd is supported or not. For any other types T and Abi, there is no member value. If the program adds specializations for std::simd_size or std::simd_size_v(since C++17), the behavior is undefined.
