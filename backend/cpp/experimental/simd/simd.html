template< class T, class Abi = simd_abi::compatible > class simd; (parallelism TS v2) The class template simd is a data-parallel type. The width of a given simd instantiation is a constant expression, determined by the template parameters. An ABI tag is a type in the simd_abi namespace that indicates a choice of size and binary representation for objects of data-parallel type.
