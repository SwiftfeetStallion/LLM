TS v2) If is_simd_mask_v is true and U is bool, or if is_simd_v is true and U is a vectorizable type, provides the member constant value equal to the minimum alignment of a pointer passed to a load constructor, copy_from function, or copy_to function. For any other types T and U, there is no member value. If the program adds specializations for std::memory_alignment or std::memory_alignment_v(since C++17), the behavior is undefined.
