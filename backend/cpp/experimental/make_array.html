Types > constexpr std::array make_array( Types&&... t ); (library fundamentals TS v2) Creates a std::array whose size is equal to the number of arguments and whose elements are initialized from the corresponding arguments. Returns std::array{std::forward(t)...}. If D is void, then the deduced type VT is std::common_type_t. Otherwise, it is D. If D is void and any of std::decay_t... is a specialization of std::reference_wrapper, the program is ill-formed.
