B > struct conjunction; (library fundamentals TS v2) Forms the logical conjunction of the type traits B..., effectively performing a logical AND on the sequence of traits. The specialization std::experimental::conjunction has a public and unambiguous base that is if sizeof...(B) == 0, std::true_type; otherwise the first type Bi in B1, ..., BN for which bool(Bi::value) == false, or BN if there is no such type. The member names of the base class, other than conjunction and operator=, are not hidden and are unambiguously available in conjunction. Conjunction is short-circuiting: if there is a template type argument Bi with bool(Bi::value) == false, then instantiating conjunction::value does not require the instantiation of Bj::value for j > i.
