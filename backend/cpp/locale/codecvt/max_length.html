Contents 1 Return value 2 Notes 3 Example 4 See also [edit] Return value The maximum number of ExternT characters that could be consumed if converted by in() to produce one InternT character. The non-converting specialization std::codecvt returns 1. [edit] Notes If the encoding is state-dependent (encoding() == -1), then more than max_length() external characters may be consumed to produce one internal character.
