This function may be used in the beginning of a subroutine that must hide the floating-point exceptions it may raise from the caller. If only some exceptions must be suppressed, while others must be reported, the non-stop mode is usually ended with a call to std::feupdateenv after clearing the unwanted exceptions. [edit] Parameters envp - pointer to the object of type std::fenv_t where the floating-point environment will be stored [edit] Return value ​0​ on success, non-zero otherwise.
