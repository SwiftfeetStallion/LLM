The default ABI tag is determined by the implementation at compile-time. 2) std::datapar::simd is an alias template that allows users to specify the width to a certain size. The default width is determined by the implementation at compile-time. Every specialization of basic_simd is a complete type. The specialization is enabled, if T is a vectorizable type, and there exists value M in the range [1, 64] such that Abi is deduce-abi-t, otherwise disabled, if T is not a vectorizable type, otherwise, it is implementation-defined if such specialization is enabled. If basic_simd is disabled, the specialization has all deleted default constructor, destructor, copy constructor, and copy assignment. In addition, only the member types below are present. If basic_simd is enabled, basic_simd is TriviallyCopyable.
