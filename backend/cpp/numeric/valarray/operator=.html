If size() != other.size(), first resizes *this as if by resize(other.size()). Each element of *this is assigned the value of the corresponding element of other. 2) Move assignment operator. Replaces the contents of *this with those of other. The value of other is unspecified after this operation. The complexity of this operation may be linear if T has non-trivial destructors, but is usually constant otherwise. 3) Replaces each value in *this with a copy of val. 4-7) Replaces the contents of *this with the result of a generalized subscripting operation. The behavior is undefined if size() does not equal the length of other or if any value on the left depends on the value on the right (e.g. v = v[v > 2]). 8) Assigns the contents of initializer list il. Equivalent to *this = valarray(il).
