If either M or N is not an integer type, or if either is (possibly cv-qualified) bool, the program is ill-formed. If either |m| or |n| is not representable as a value of type std::common_type_t, the behavior is undefined. Otherwise, returns the greatest common divisor of |m| and |n|. [edit] Exceptions Throws no exceptions.
