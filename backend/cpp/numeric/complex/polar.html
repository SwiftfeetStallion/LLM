The behavior is undefined if r is negative or NaN, or if theta is infinite. Contents 1 Parameters 2 Return value 3 Notes 4 Example 5 Defect reports 6 See also [edit] Parameters r - magnitude theta - phase angle [edit] Return value A complex number determined by r and theta. [edit] Notes std::polar(r, theta) is equivalent to any of the following expressions: r * std::exp(theta * 1i) r * (cos(theta) + sin(theta) * 1i) std::complex(r * cos(theta), r * sin(theta)). Using polar instead of exp can be about 4.5x faster in vectorized loops.
