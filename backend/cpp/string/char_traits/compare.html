The comparison is done lexicographically. If count is zero, strings are considered equal. See CharTraits for the general requirements on character traits for X::compare. [edit] Parameters s1, s2 - pointers to character strings to compare count - the number of characters to compare from each character string [edit] Return value Negative value if s1 is less than s2. ​0​ if s1 is equal to s2. Positive value if s1 is greater than s2. [edit] Complexity Linear in count.
