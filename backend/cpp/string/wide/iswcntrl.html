If the value of ch is neither representable as a wchar_t nor equal to the value of the macro WEOF, the behavior is undefined. [edit] Notes ISO 30112 defines POSIX control characters as Unicode characters U+0000..U+001F, U+007F..U+009F, U+2028, and U+2029 (Unicode classes Cc, Zl, and Zp).
