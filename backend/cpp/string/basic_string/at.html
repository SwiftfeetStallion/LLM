C++20) Returns a reference to the character at specified location pos. Bounds checking is performed, exception of type std::out_of_range will be thrown on invalid access. Contents 1 Parameters 2 Return value 3 Exceptions 4 Complexity 5 Example 6 Defect reports 7 See also [edit] Parameters pos - position of the character to return [edit] Return value Reference to the requested character. [edit] Exceptions Throws std::out_of_range if pos >= size(). If an exception is thrown for any reason, these functions have no effect (strong exception safety guarantee). [edit] Complexity Constant.
