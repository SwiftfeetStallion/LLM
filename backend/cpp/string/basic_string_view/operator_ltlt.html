After constructing and checking the sentry object, determines the output format padding. Then stores each character from the resulting sequence seq (the contents of v with padding) to the output stream os as if by calling os.rdbuf()->sputn(seq, n), where n is std::max(os.width(), str.size()). Finally, calls os.width(0) to cancel the effects of std::setw, if any.
