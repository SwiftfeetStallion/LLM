The terminating null character is considered to be a part of the string and can be found if searching for '\0'. Contents 1 Parameters 2 Return value 3 Example 4 See also [edit] Parameters str - pointer to the null-terminated byte string to be analyzed ch - character to search for [edit] Return value Pointer to the found character in str, or a null pointer if no such character is found.
