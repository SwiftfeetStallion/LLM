Args > void emplace( Args&&... args ); (since C++11) Pushes a new element to the priority queue. The element is constructed in-place, i.e. no copy or move operations are performed. The constructor of the element is called with exactly the same arguments as supplied to the function. Effectively calls c.emplace_back(std::forward(args)...); std::push_heap(c.begin(), c.end(), comp); Contents 1 Parameters 2 Return value 3 Complexity 4 Example 5 See also [edit] Parameters args - arguments to forward to the constructor of the element [edit] Return value (none) [edit] Complexity Logarithmic number of comparisons plus the complexity of Container::emplace_back.
