Args > void emplace( Args&&... args ); (since C++11) (until C++17) template< class... Args > decltype(auto) emplace( Args&&... args ); (since C++17) Pushes a new element to the end of the queue. The element is constructed in-place, i.e. no copy or move operations are performed. The constructor of the element is called with exactly the same arguments as supplied to the function. Effectively calls c.emplace_back(std::forward(args)...);.
