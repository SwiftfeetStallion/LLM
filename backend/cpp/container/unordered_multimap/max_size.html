(constexpr since C++26) Returns the maximum number of elements the container is able to hold due to system or library implementation limitations, i.e. std::distance(begin(), end()) for the largest container. Contents 1 Return value 2 Complexity 3 Notes 4 Example 5 See also [edit] Return value Maximum number of elements. [edit] Complexity Constant. [edit] Notes This value typically reflects the theoretical limit on the size of the container, at most std::numeric_limits::max(). At runtime, the size of the container may be limited to a value smaller than max_size() by the amount of RAM available.
