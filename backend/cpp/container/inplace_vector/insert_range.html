This section is incomplete Each iterator in the range rg is dereferenced exactly once. If rg overlaps with *this, the behavior is undefined. Contents 1 Parameters 2 Return value 3 Exceptions 4 Example 5 See also [edit] Parameters pos - iterator before which the content will be inserted (pos may be the end() iterator) rg - a container compatible range, that is, an input_range whose elements are convertible to T Type requirements -If any of the following conditions is satisfied, the behavior is undefined: T is not EmplaceConstructible into inplace_vector from *ranges::begin(rg). T is not MoveInsertable into inplace_vector. T does not satisfy the requirements of MoveConstructible, MoveAssignable, or Swappable. [edit] Return value An iterator to the first element inserted into *this, or pos if rg is empty. Exceptions std::bad_alloc, if ranges::distance(rg) + size() > capacity(). The elements of *this are not modified. Any exception thrown by insertion (i.e. by copy/move constructor, move/copy assignment operator of T) or by any LegacyInputIterator operation. The elements of *this in the range [​0​, pos) are not modified.
