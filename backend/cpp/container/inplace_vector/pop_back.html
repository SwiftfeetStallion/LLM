If empty() is true: If the implementation is hardened, a contract violation occurs. Moreover, if the contract-violation handler returns under “observe” evaluation semantic, the behavior is undefined. If the implementation is not hardened, the behavior is undefined. Iterators and references to the last element are invalidated. The end() iterator is also invalidated. [edit] Complexity Constant.
