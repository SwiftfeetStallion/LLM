(since C++26) Specializes the std::swap algorithm for std::inplace_vector. Swaps the contents of lhs and rhs. Calls lhs.swap(rhs). Contents 1 Parameters 2 Complexity 3 Exceptions 4 Example 5 See also [edit] Parameters lhs, rhs - containers whose contents to swap [edit] Complexity Linear in size of the containers. Exceptions noexcept specification:   noexcept(N == 0 ||          (std::is_nothrow_swappable_v && std::is_nothrow_move_constructible_v)) [edit] Example Run this code #include #include #include   int main() { std::inplace_vector alice{1, 2, 3}; std::inplace_vector bob{7, 8, 9, 10};   auto print = [](const int& n) { std::cout << ' ' << n; };   // Print state before swap std::cout << "Alice:"; std::for_each(alice.begin(), alice.end(), print); std::cout << "\nBobby:"; std::for_each(bob.begin(), bob.end(), print); std::cout << '\n';   std::cout << "-- SWAP\n"; std::swap(alice, bob);   // Print state after swap std::cout << "Alice:"; std::for_each(alice.begin(), alice.end(), print); std::cout << "\nBobby:"; std::for_each(bob.begin(), bob.end(), print); std::cout << '\n'; } Output: Alice: 1 2 3 Bobby: 7 8 9 10 -- SWAP Alice: 7 8 9 10 Bobby: 1 2 3 [edit] See also swap swaps the contents (public member function) [edit]
