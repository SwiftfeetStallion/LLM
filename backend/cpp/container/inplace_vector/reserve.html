The request to increase the capacity (i.e., the internal storage size) is ignored because std::inplace_vector is a fixed-capacity container. [edit] Exceptions std::bad_alloc if new_cap > capacity() is true. [edit] Notes This function exists for compatibility with vector-like interfaces.
