(1) (since C++26) constexpr reference unchecked_push_back( T&& value ); (2) (since C++26) Appends the given element value to the end of the container. Equivalent to: return *try_push_back(std::forward(value)); 1) The new element is initialized as a copy of value. 2) value is moved into the new element. Before the call to these functions size() < capacity() must be true. Otherwise, the behavior is undefined. No iterators or references are invalidated, except end(), which is invalidated if the insertion occurs.
