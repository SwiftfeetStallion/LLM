Args > iterator emplace_hint( const_iterator hint, Args&&... args ); (since C++23) (constexpr since C++26) Inserts a new element into the container as close as possible to the position just before hint. The constructors of the key and mapped value are called with exactly the same arguments as supplied to the function, forwarded with std::forward(args)....
