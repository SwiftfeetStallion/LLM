at_quick_exit( /*c-atexit-handler*/* func ) noexcept; (1) (since C++11) extern "C++" using /*atexit-handler*/ = void(); extern "C" using /*c-atexit-handler*/ = void(); (2) (exposition only*) Registers the function pointed to by func to be called on quick program termination (via std::quick_exit). Calling the function from several threads does not induce a data race. The implementation is guaranteed to support the registration of at least 32 functions. The exact limit is implementation-defined. The registered functions will not be called on normal program termination. If a function need to be called in that case, std::atexit must be used. [edit] Notes The two overloads are distinct because the types of the parameter func are distinct (language linkage is part of its type).
