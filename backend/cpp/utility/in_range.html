It is a compile-time error if either T or U is a non-integer type, a character type, or bool. [edit] Possible implementation template constexpr bool in_range(T t) noexcept { return std::cmp_greater_equal(t, std::numeric_limits::min()) && std::cmp_less_equal(t, std::numeric_limits::max()); } [edit] Notes This function cannot be used with enums (including std::byte), char, char8_t, char16_t, char32_t, wchar_t and bool.
