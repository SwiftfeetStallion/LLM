Contents 1 Parameters 2 Return value 3 Example 4 See also [edit] Parameters (none) [edit] Return value If this object represents a position in a body of a function, returns an implementation-defined null-terminated byte string corresponding to the name of the function. Otherwise, an empty string is returned.
