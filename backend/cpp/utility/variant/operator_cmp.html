The contained values are compared (using the corresponding operator of T) only if both lhs and rhs contain values corresponding to the same index. Otherwise, lhs is considered equal to rhs if, and only if, both lhs and rhs do not contain a value. lhs is considered less than rhs if, and only if, either rhs contains a value and lhs does not, or lhs.index() is less than rhs.index(). 1-6) Let @ denote the corresponding comparison operator, for each of these functions: If, for some values of I, the corresponding expression GET ﻿(lhs) @ GET ﻿(rhs) is ill-formed or its result is not convertible to bool, the program is ill-formed.
