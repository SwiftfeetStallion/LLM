[edit] Semantic requirements If T is an object type, then copy_constructible is modeled only if given v, an lvalue of type (possibly const) T or an rvalue of type const T, the following are true: After the definition T u = v;, u is equal to v and v is not modified; T(v) is equal to v and does not modify v.
